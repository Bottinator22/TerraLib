# TerraLib
Starbound library mod.
Though initially named after its Terraria-like features, it's more than that now.

`terraliblite` - The core part of TerraLib. Included in both TerraLib and TerraLib Lite on the workshop.  
`terralib` - Included in TerraLib only. Contains non-vanilla-compatible components of TerraLib.  
`wormlib` - Included in both TerraLib and TerraLib Lite. Exists for backwards compatibility purposes.
